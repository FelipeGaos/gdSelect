GDSelect - Demo

Una librería de select personalizado con búsqueda y selección múltiple.

Select Básico

const basicSelect = new GDSelect('#basic-select', {
    placeholder: 'Seleccione una opción',
    search: true
});

Select Múltiple

const multipleSelect = new GDSelect('#multiple-select', {
    placeholder: 'Seleccione múltiples opciones',
    search: true,
    multiple: true
});

Select con Opciones Deshabilitadas

const disabledSelect = new GDSelect('#disabled-select', {
    placeholder: 'Seleccione una opción',
    search: true
});

Métodos de la API

const apiSelect = new GDSelect('#api-select', {
    placeholder: 'Seleccione una opción',
    search: true
});

// Establecer valor
apiSelect.setValue('2');

// Obtener valor
const value = apiSelect.getValue();
